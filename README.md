# 1 项目介绍
基于SSM+Vue的租房网站：后端 SpringBoot、Mybatis-Plus，前端Vue+ElementUI，具体功能如下：
## 管理端
- 个人中心：查看和修改个人信息、修改密码
- 用户管理
- 房屋类型管理
- 房屋信息管理
- 房屋评价管理
- 在线签约管理
- 轮播图管理
- 在线客服
## 用户端
- 首页
- 房屋信息
- 房屋资讯
- 交流论坛
- 在线客服
- 个人中心
